# Anime Suggestions

A simple anime suggestion web app that recommends titles based on your selected genre and explains **why** each suggestion is worth watching.

## Features
- Pick from multiple genres (Action, Romance, Fantasy, Comedy, Slice of Life, Thriller).
- Get two recommendations each time you click **Suggest Anime**.
- See a short reason for each recommendation.

## Run locally
Because this app uses plain HTML/CSS/JavaScript, you can run it with any static server.

```bash
python3 -m http.server 8000
```

Then open: `http://localhost:8000`
